Detailed Description

Evolutionary Algorithm Class.

All methods involved in the evolutionary algorithm. Parent selection operator (rank based), fitness functions, and genetic operators.

This is Conventional Neuro-Evolution (CNE) that operates only on the weights of the agent ANNs. Parent selection is rank based. Fitness is based on the energy level of an agent, mutation is done via random perturbations of connection weight values, crossover is single-point.
